# First-Home-Lab

## First recorded project ever, Home Lab built for understanding computer networking, cybersecurity, RF, and all things IT.

## First order of business - Set up hardware and understand the very basics of IP's, Networking and Terminal language.

## Setup 
1. TP-Link BE3600 Archer Router with Dual-Band Capability (5Ghz and 2.4Ghz) Wifi 7
2. TP-Link 8 port Unmanaged switch for expansion
3. Tons of ethernet cables + Ethernet to USB-C adapter
4. Macbook Air and IPhone

## Current Network Config
1. No internet, Wi-Fi off, ethernet only for connectivity
