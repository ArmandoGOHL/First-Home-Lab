# First-Home-Lab

## First recorded project ever, Home Lab built for understanding computer networking, cybersecurity, RF, and all things IT.

## First order of business - Set up hardware and understand the very basics of IP's, Networking and Terminal language.

## Setup 
-TP-Link BE3600 Archer Router with Dual-Band Capability (5Ghz and 2.4Ghz) Wifi 7
-TP-Link 8 port Unmanaged switch for expansion
-Tons of ethernet cables + Ethernet to USB-C adapter
-Macbook Air and IPhone

## Current Network Config
-No internet, Wi-Fi off, ethernet only for connectivity
